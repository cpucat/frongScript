How to use:

python3 is needed

go to the frongScript folder in a command window

run: "python3 ./frongpy/__main__.py --help" to test the program

to compile and run hello world: "python3 ./frongpy/__main__.py --file ./examples/hello_frong.frn"


for documentation visit: "https://esolangs.org/wiki/Chicken" just replace "chicken" with "frong"
